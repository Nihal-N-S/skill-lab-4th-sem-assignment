# Student API (Node.js)

This is a simple Node.js project to manage student data using HTTP methods.

## Features

* Get all students
* Get student by ID
* Add new student
* Update student
* Delete student

## How to Run

1. Install Node.js

2. Open terminal in project folder

3. Run:
   node server.js

4. Server will start on:
   http://localhost:4000

## API Endpoints

* GET /students
* GET /students/:id
* POST /students
* PUT /students/:id
* DELETE /students/:id

## Note

Data is stored temporarily in memory, so it will reset when the server restarts.
